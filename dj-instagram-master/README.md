# Install
- Clone this repo
- Activate your virtualenv
- Navigate to this project's directory in your computer
- Install requirements
    - `$ pip install -r requirements.txt`
- Update your database
    - `$ python manage.py migrate`
- You're all set!
    - `$ python manage.py runserver`
